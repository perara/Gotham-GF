    Gotham.Preload Class

    Defined in: src\Modules\Preload.coffee:2
    Module: Framework
    Parent Module: Framework

    Preload Class This class contains storage for audio, video, image and json It keeps track of current loaded object, and the total number of objects It can serve these preloaded files on the fly.

    Constructor

    Gotham.Preload

    ()

    Defined in src\Modules\Preload.coffee:2

    Example:

    # Examples
                                # Preloading a Image file
                                Gotham.Preload.image("/assets/img/settings_close.png", "settings_close", "image")
                                # Preloading a JSON File
                                Gotham.Preload.json("/assets/json/json.json", "map")
                                # Fetching the JSON file
                                Gotham.Preload.fetch("map", "json")
                                # Fetching the image file
                                Gotham.Preload.fetch("settings_close", "image")
